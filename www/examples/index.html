Cooperative map example

Drop markers on a map that any user can move. Synchronize the map continuously or on click.

Cooperative shopping list tutorial

Edit shopping list items and quantities cooperatively. See where other users have focus based on a highlight. This implementation of this demo is described in detail in the Creating a cooperative shopping list tutorial in the OpenCoweb docs.

Tip: Add ?cowebkey=anything to a demo URL to join a unique session keyed by "anything" using the app.
